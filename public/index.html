What is dotf?

dotf is a modern dotfiles management tool that helps you:

  • Sync your configuration files across multiple machines
  • Track changes to your dotfiles with git
  • Easily backup and restore your development environment
  • Share your configurations with others

Installation Process

The installer script will:

  • Download the latest binary for your platform
  • Install it to /usr/local/bin (requires sudo)
  • Verify the installation
  • Display usage instructions

System Requirements

  • Linux or macOS
  • curl (for downloading)
  • sudo access (for installation)
  • git (for dotfiles management)
